Este repositorio contiene un sistema de registro de usuarios mediante inicio de sesión utilizando Firebase Firestore como base de datos. A continuación, describiré cómo funciona y qué características ofrece: Inicio de Sesión: Los usuarios pueden iniciar sesión con las siguientes credenciales: Usuario: usuario / contraseña: usuario123 Administrador: admin / contraseña: admin123 El acceso está restringido según el rol del usuario. Panel de Administrador: Si inicias sesión como administrador, tendrás acceso a: Un formulario para registrar nuevos usuarios. Una tabla para mostrar los datos de los usuarios registrados. Operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre los usuarios. Registro de Usuarios: Si inicias sesión como usuario, podrás: Completar un formulario para registrar tus datos personales. Al finalizar el registro, recibirás un mensaje de éxito. Validación de Datos: El sistema valida los datos ingresados en el inicio de sesión y el registro. Si los datos no son correctos, no se permitirá el acceso a ninguna de las dos páginas. Configuración Firebase Firestore: La base de datos se encuentra en Firestore. Para editar la base de datos, debes hacerlo directamente desde Firestore, ya que el CRUD es local.
