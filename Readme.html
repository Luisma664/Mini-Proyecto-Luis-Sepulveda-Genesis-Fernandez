El proyecto consiste en la implementación de un sistema de gestión de usuarios a través de un formulario en un sitio web. El sitio web consta de tres páginas principales: una página de inicio de sesión con roles de "admin" y "usuario", donde las credenciales para el administrador son "admin123" y para el usuario son "usuario123". Una vez autenticado, el usuario puede acceder al formulario para registrar sus datos, incluyendo nombre, apellido, cédula y calificaciones en materias específicas como matemáticas, inglés y química. Por otro lado, el administrador tiene acceso privilegiado para visualizar la lista de usuarios registrados, editar sus datos, agregar nuevos usuarios o eliminar registros existentes. Además, el administrador puede realizar operaciones de lectura, escritura, actualización y eliminación (CRUD) sobre la base de datos de usuarios. Este proyecto busca facilitar la gestión eficiente de usuarios a través de un sistema web intuitivo y seguro, brindando a los administradores la capacidad de administrar y supervisar la información de los usuarios de manera efectiva.
